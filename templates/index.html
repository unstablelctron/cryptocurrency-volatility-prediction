Crypto Volatility Predictor

This is a Flask-powered API for your trained volatility prediction model. Use the endpoint /api/predict to get predictions.

API Usage

URL: POST /api/predict

Content-Type: application/json

Expected JSON Keys (example)

    {% for col in feature_columns %}
  • {{ col }}
    • {% endfor %}

Example request body for a single record (adapt to your actual feature names/values): 

{
  "open": 51.37,
  "high": 57.85,
  "low": 49.62,
  "close": 51.31,
  "volume": 3132405,
  "marketCap": 123456789,
  "tr": 2.31,
  "atr_14": 1.12,
  "log_return": -0.0123,
  "crypto_name_encoded": 0,
  "Day": 15,
  "Month": 5,
  "Year": 2024
}

You can also send an array of such objects to get batch predictions.
